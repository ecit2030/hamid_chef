## 🚀 الخطوة 1: الوصول إلى لوحة التحكم

1. افتح المتصفح وانتقل إلى: `http://localhost/admin/discount-codes`
2. قم بتسجيل الدخول كمشرف
3. ستظهر لك صفحة "أكواد الخصم" فارغة (لأنه لا توجد أكواد بعد)

## 📝 الخطوة 2: إنشاء أول كود خصم

### الطريقة 1: عبر لوحة التحكم (موصى بها)

1. اضغط على زر "إضافة كود خصم"
2. املأ البيانات:
    - **الكود**: `SUMMER2026` (يجب أن يكون فريداً)
    - **الوصف**: `خصم صيفي 20%`
    - **نوع الخصم**: نسبة مئوية
    - **قيمة الخصم**: `20`
    - **الحد الأدنى للطلب**: `100` (اختياري)
    - **الحد الأقصى للخصم**: `200` (اختياري)
    - **تاريخ البداية**: `2026-01-01`
    - **تاريخ النهاية**: `2026-12-31`
    - **عدد الاستخدامات الكلي**: `1000` (اختياري - اتركه فارغاً لعدد غير محدود)
    - **عدد الاستخدامات لكل مستخدم**: `3`
    - **نشط**: ✅ (مفعّل)
3. اضغط "حفظ"

### الطريقة 2: عبر قاعدة البيانات

إذا كنت تفضل إنشاء الكود مباشرة في قاعدة البيانات:

```sql
    '2026-12-31 23:59:59',

```

## 🧪 الخطوة 3: اختبار الكود

### اختبار عبر API (Postman)

#### 1. الحصول على Token

```http
POST http://localhost/api/login
Content-Type: application/json

{
  "phone_number": "777777777",
  "password": "password"
}
```

احفظ الـ `token` من الـ response.

#### 2. التحقق من الكود

```http
POST http://localhost/api/discount-codes/validate
Authorization: Bearer {your_token}
Content-Type: application/json

{
  "code": "SUMMER2026",
  "amount": 500
}
```

**Response المتوقع:**

```json
{
    "success": true,
    "data": {
        "valid": true,
        "discount_code_id": 1,
        "code": "SUMMER2026",
        "type": "percentage",
        "value": 20,
        "original_amount": 500,
        "discount_amount": 100,
        "final_amount": 400

    "message": "الكود صالح"
}
```

## 📊 الخطوة 4: مراجعة الإحصائيات

1. ارجع إلى لوحة التحكم: `http://localhost/admin/discount-codes`
2. اضغط على "عرض" بجانب الكود
3. ستشاهد:
    - إجمالي الاستخدامات
    - إجمالي مبلغ الخصم
    - عدد المستخدمين الفريدين
    - الاستخدامات المتبقية
    - قائمة بآخر 10 استخدامات

## ❓ استكشاف الأخطاء

### المشكلة: "الكود غير موجود"

- تأكد من كتابة الكود بشكل صحيح (حساس لحالة الأحرف)
- تحقق من أن الكود موجود في قاعدة البيانات

### المشكلة: "الكود غير نشط"

- تأكد من تفعيل الكود في لوحة التحكم
- تحقق من حقل `is_active = 1` في قاعدة البيانات

### المشكلة: "الكود منتهي الصلاحية"

- تحقق من تاريخ البداية والنهاية
- تأكد من أن التاريخ الحالي بين `start_date` و `end_date`

### المشكلة: "تم استنفاد عدد استخدامات الكود"

- تحقق من `usage_count` و `usage_limit`
- يمكنك زيادة `usage_limit` أو جعله `NULL` لعدد غير محدود

### المشكلة: "لقد استخدمت هذا الكود من قبل"

- تحقق من `per_user_limit`
- يمكنك زيادة العدد المسموح لكل مستخدم

### المشكلة: "الحد الأدنى للطلب هو X ريال"

- تأكد من أن مبلغ الطلب أكبر من أو يساوي `min_order_amount`
