# عرض معلومات الخصم في شاشات الحجز

## الملخص

تم تحديث شاشات الحجز لعرض معلومات الخصم عندما يتم تطبيق كود خصم على الحجز.

## التغييرات المنفذة

### 1. تحديث صفحة تفاصيل الحجز (ShowBooking.vue)

**الملف:** `resources/js/Components/Chef/booking/ShowBooking.vue`

**التحسينات:**

- إضافة قسم "معلومات الخصم" في تفاصيل التسعير
- عرض المجموع الفرعي (السعر قبل الخصم)
- عرض كود الخصم المستخدم في badge أخضر
- عرض مبلغ الخصم بلون أخضر
- عرض المبلغ الإجمالي النهائي بعد الخصم

**مثال العرض:**

```
السعر الأساسي: 500.00 ر.س
عدد الساعات (4): 2,000.00 ر.س
الضيوف الإضافيون (2): 100.00 ر.س
─────────────────────────────
المجموع الفرعي: 2,100.00 ر.س
الخصم [TEST2024]: -210.00 ر.س
─────────────────────────────
المبلغ الإجمالي: 1,890.00 ر.س
```

### 2. تحديث قائمة الحجوزات (ShowBookings.vue)

**الملف:** `resources/js/Components/Chef/booking/ShowBookings.vue`

**التحسينات:**

- إضافة مؤشر خصم (discount badge) بجانب المبلغ الإجمالي
- عرض أيقونة "+" مع نص "الخصم" عند تطبيق خصم
- Badge أخضر صغير يظهر فقط عند وجود خصم

**مثال العرض في البطاقة:**

```
المبلغ الإجمالي
1,890.00 ر.س [+ الخصم]
```

### 3. تحديث BookingDTO

**الملف:** `app/DTOs/BookingDTO.php`

**التحسينات:**

- إضافة حقل `discount_code` لتخزين كود الخصم
- تحديث constructor لقبول `discount_code`
- تحديث `fromModel()` لجلب كود الخصم من العلاقة
- تحديث `toArray()` و `toIndexArray()` لتضمين معلومات الخصم
- إضافة علاقة `discountCodeRelation` للوصول إلى كائن الخصم الكامل

### 4. تحديث BookingRepository

**الملف:** `app/Repositories/BookingRepository.php`

**التحسينات:**

- إضافة `'discountCode'` إلى `$defaultWith` لتحميل علاقة الخصم تلقائياً
- يضمن جلب معلومات كود الخصم مع كل استعلام للحجوزات

### 5. إضافة الترجمات العربية

**الملف:** `resources/js/locales/ar.json`

**الترجمات المضافة:**

```json
{
    "booking": {
        "subtotal": "المجموع الفرعي",
        "discount": "الخصم",
        "discount_applied": "تم تطبيق الخصم",
        "discount_code": "كود الخصم"
    }
}
```

## كيفية الاختبار

### 1. إنشاء حجز بخصم

```bash
# استخدم API لإنشاء حجز مع كود خصم
POST /api/bookings
{
  "chef_service_id": 1,
  "date": "2026-02-10",
  "start_time": "14:00",
  "hours_count": 4,
  "number_of_guests": 6,
  "discount_code": "TEST2024"
}
```

### 2. عرض تفاصيل الحجز

- انتقل إلى صفحة تفاصيل الحجز
- تحقق من ظهور قسم الخصم في تفاصيل التسعير
- تأكد من عرض كود الخصم والمبلغ المخصوم

### 3. عرض قائمة الحجوزات

- انتقل إلى صفحة قائمة الحجوزات
- ابحث عن الحجوزات التي تحتوي على خصم
- تحقق من ظهور badge "الخصم" بجانب المبلغ الإجمالي

## الحقول المعروضة

### في صفحة التفاصيل:

1. **المجموع الفرعي** - السعر قبل الخصم (original_amount)
2. **كود الخصم** - الكود المستخدم (discount_code) في badge أخضر
3. **مبلغ الخصم** - المبلغ المخصوم (discount_amount) بلون أخضر
4. **المبلغ الإجمالي** - السعر النهائي بعد الخصم (total_amount)

### في قائمة الحجوزات:

1. **المبلغ الإجمالي** - السعر النهائي (total_amount)
2. **مؤشر الخصم** - badge صغير يظهر عند وجود خصم

## الشروط

- يظهر قسم الخصم فقط عندما: `discount_amount > 0`
- يظهر كود الخصم فقط عندما: `discount_code` موجود
- يتم حساب المجموع الفرعي من: `original_amount` أو `total_amount` إذا لم يكن موجوداً
